Input - BitRabbit Passport UI

输入框是最基础的组件

  • 普通元素
  • 无需JS
  • 锚定选择器: .bpc-input

尺寸：配置项bpc-size

<input type="text" class="bpc-input" placeholder="Large Size" bpc-size="large">
<input type="text" class="bpc-input" placeholder="Normal Size">
<input type="text" class="bpc-input" placeholder="Small Size" bpc-size="small">

禁止状态：配置项disabled

    <input type="text" class="bpc-input" disabled placeholder="Large Size" bpc-size="large">
    <input type="text" class="bpc-input" disabled placeholder="Normal Size">
    <input type="text" class="bpc-input" disabled placeholder="Small Size" bpc-size="small">

配置

配置项 原生 描述
bpc-size N 尺寸大小，取值空, large, small
disabled N 禁用输入框
