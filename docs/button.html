Button - BitRabbit Passport UI

按钮是最基本的元素

  • 普通元素
  • 无需JS
  • 锚定选择器: .bpc-btn

基本使用：配置项bpc-type

<button class="bpc-btn" bpc-type="primary">Primary</button>
<button class="bpc-btn">Default</button>
<button class="bpc-btn" bpc-type="dotted">Dotted</button>
<button class="bpc-btn" bpc-type="danger">Danger</button>

尺寸：配置项bpc-size

<button class="bpc-btn" bpc-size="large">Large</button>
<button class="bpc-btn">Normal</button>
<button class="bpc-btn" bpc-size="small">Small</button>

图标按钮：配置项bpc-icon

<button class="bpc-btn" bpc-type="primary" bpc-icon><i class="iconfont icon-search"></i></button>
<button class="bpc-btn" bpc-icon><i class="iconfont icon-search"></i></button>
<button class="bpc-btn" bpc-type="dotted" bpc-icon><i class="iconfont icon-search"></i></button>
<button class="bpc-btn" bpc-type="danger" bpc-icon><i class="iconfont icon-search"></i></button>

禁止状态：原生配置项disabled

<button class="bpc-btn" disabled bpc-type="primary">Primary</button>
<button class="bpc-btn" disabled>Default</button>
<button class="bpc-btn" disabled bpc-type="dotted">Dotted</button>
<button class="bpc-btn" disabled bpc-type="danger">Danger</button>
<button class="bpc-btn" disabled bpc-type="primary" bpc-icon><i class="iconfont icon-search"></i></button>
<button class="bpc-btn" disabled bpc-icon><i class="iconfont icon-search"></i></button>
<button class="bpc-btn" disabled bpc-type="dotted" bpc-icon><i class="iconfont icon-search"></i></button>
<button class="bpc-btn" disabled bpc-type="danger" bpc-icon><i class="iconfont icon-search"></i></button>

配置

配置项 原生 描述
disabled Y 是否禁止按钮
bpc-type N 类型，取值空, primary, dotted, danger
bpc-size N 尺寸大小，取值空, large, small
bpc-icon N 是否为图标按钮
bpc-icon N 是否为图标按钮
