Select - BitRabbit Passport UI

选择器

  • 脚本元素
  • 需要JS
  • 锚定选择器: .bpc-select

基本使用

<select class="bpc-select">
  <option value="1">A</option>
  <option value="2" selected>B</option>
  <option value="3" disabled>C</option>
  <option value="4">D</option>
</select>

尺寸：配置项bpc-size

<select class="bpc-select" bpc-size="large">
  <option value="1">A</option>
  <option value="2" selected>B</option>
  <option value="3" disabled>C</option>
  <option value="4">D</option>
</select>

配置

配置项 原生 描述
bpc-size N 尺寸大小，取值空, large, small
disabled N 禁用选择框
