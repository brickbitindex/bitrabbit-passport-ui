Pagination - BitRabbit Passport UI

分页器是数据呈现部分最常用的组件之一。在分页器的设计里，我们完全使用JS来生成DOM，因为我们不认为在ServerSide Render里让后端模板来拼接分页器的所有逻辑是不明智的。

  • 脚本元素
  • 需要JS
  • 锚定选择器: .bpc-pagination

简单使用

<div class="bpc-pagination" bpc-total-page="20" bpc-current-page="16" bpc-url-template="/?p={index}"></div>

尺寸：配置项bpc-size

<div class="bpc-pagination" bpc-total-page="20" bpc-current-page="1" bpc-url-template="/?p={index}" bpc-size="large"></div>
<div class="bpc-pagination" bpc-total-page="20" bpc-current-page="16" bpc-url-template="/?p={index}"></div>
<div class="bpc-pagination" bpc-total-page="20" bpc-current-page="20" bpc-url-template="/?p={index}" bpc-size="small"></div>

配置

配置项 原生 描述
bpc-size N 尺寸大小，取值空, large, small
bpc-total-page N 总共几页
bpc-current-page N 当前第几页
bpc-url-template N 点击各链接的url模板，使用{index}来代表页码
bpc-target N 链接的a元素的target属性，若在新页面打开，则此项为_blank
