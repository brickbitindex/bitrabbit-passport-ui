Limited Input - BitRabbit Passport UI

受限输入框，主要是为了类似验证码之类的，限制长度的输入框。

  • 脚本元素
  • 需要JS
  • 锚定选择器: .bpc-limited-input

限制长度：配置项bpc-length

<input class="bpc-limited-input" bpc-length="5">
<input class="bpc-limited-input" bpc-length="6">
<input class="bpc-limited-input" bpc-length="7">

尺寸：配置项bpc-size

<input class="bpc-limited-input" bpc-length="5">
<input class="bpc-limited-input" bpc-length="5" bpc-size="large">
<input class="bpc-limited-input" bpc-length="5" bpc-size="small">

配置

配置项 原生 描述
bpc-size N 尺寸大小，取值空, large, small
disabled N 禁用输入框
