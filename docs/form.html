Form - BitRabbit Passport UI

表单是最常用的模式，Form的实践在设计时考虑了绝大部分需求的情况，我们不认为实践需要满足所有需求。在Form的模式里，整个Form需要呈现出List的特质，所以选择了ul元素，内部每个li元素为一个表单项。

  • 普通元素
  • 无需JS
  • 锚定选择器: ul.bpc-form

简单表单

<ul class="bpc-form">
  <li>
    <input type="text" class="bpc-input" placeholder="Username">
  </li>
  <li>
    <input type="password" class="bpc-input" placeholder="password">
  </li>
  <ul>
    <li>
      <span class="bpc-checkbox-group">
        <input type="checkbox" name="g3" value="1" bpc-text="Remamber Me" checked>
      </span>
    </li>
    <li>
      <button class="bpc-btn" bpc-type="primary">Login</button>
    </li>
    <li>
      <button class="bpc-btn">Cancel</button>
    </li>
  </ul>
</ul>

在Form中再次添加ul将给予二级表单的呈现，它的间距将较一级表单更加小一点。

带标签的表单布局

Form可以通过设置bpc-label-layout来进行带标签的布局，标签的class为bpc-label，拥有一个required的属性，来表示必要的表单项。bpc-label-layout有3种可选的布局方式：horizontal、vertical和inline。可以为每个选项的li加上`bpc-error-msg="错误提示"`来产生错误提示。
