Form - BitRabbit Passport UI

表单是最常用的模式，Form的实践在设计时考虑了绝大部分需求的情况，我们不认为实践需要满足所有需求。在Form的模式里，整个Form需要呈现出List的特质，所以选择了ul元素，内部每个li元素为一个表单项。

  • 普通元素
  • 无需JS
  • 锚定选择器: ul.bpc-form

简单表单

<ul class="bpc-form">
  <li>
    <input type="text" class="bpc-input" placeholder="Username">
  </li>
  <li>
    <input type="password" class="bpc-input" placeholder="password">
  </li>
  <ul>
    <li>
      <span class="bpc-checkbox-group">
        <input type="checkbox" name="g3" value="1" bpc-text="Remamber Me" checked>
      </span>
    </li>
    <li>
      <button class="bpc-btn" bpc-type="primary">Login</button>
    </li>
    <li>
      <button class="bpc-btn">Cancel</button>
    </li>
  </ul>
</ul>
.exp2 {
  width: 360px;
}
.exp2 button {
  width: 100%;
}

在Form中再次添加ul将给予二级表单的呈现，它的间距将较一级表单更加小一点。

带标签的表单

  • Country
  • Username
  • Password 
<ul class="bpc-form" bpc-has-label>
  <li>
    <span class="bpc-label" required>Country</span>
    <select class="bpc-select">
      <option value="1">China</option>
      <option value="2">USA</option>
      <option value="3">Japan</option>
      <option value="4">France</option>
    </select>
  </li>
  <li>
    <span class="bpc-label" required>Username</span>
    <input type="text" class="bpc-input">
  </li>
  <li>
    <span class="bpc-label" required>Password</span>
    <input type="password" class="bpc-input">
  </li>
  <ul>
    <li>
      <span class="bpc-checkbox-group">
        <input type="checkbox" value="1" bpc-text="I agree the user policy">
      </span>
    </li>
    <li>
      <button class="bpc-btn" bpc-type="primary">Register</button>
      <button class="bpc-btn">Cancel</button>
    </li>
  </ul>
</ul>
.exp3 {
  width: 500px;
}
.exp3 button {
  margin-right: 8px;
}

使用bpc-has-label属性可以要求Form带标签，标签的class为bpc-label，拥有一个required的属性，来表示必要的表单项。
