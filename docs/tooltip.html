Tooltip - BitRabbit Passport UI

提示是很常见的功能。只需要在元素上加入bpc-tooltip="提示"即可。

  • 脚本元素
  • 需要JS
  • 锚定选择器: [bpc-tooltip]

基本使用

Hover Me 
<span bpc-tooltip="I am a span">Hover Me</span>
<button class="bpc-btn" bpc-tooltip="I am a button">Hover Me</button>
<select class="bpc-select" bpc-tooltip="I am a select">
  <option value="1">A</option>
  <option value="2" selected>B</option>
  <option value="3" disabled>C</option>
  <option value="4">D</option>
</select>

指定class：配置项bpc-tooltip-class

Hover Me 
<span bpc-tooltip="I am a span" bpc-tooltip-class="my-tooltip">Hover Me</span>
.my-tooltip .bpc-tooltip-content {
  color: red;
}

配置

配置项 原生 描述
bpc-tooltip N 提示文本
bpc-tooltip-class N 提示框指定class
