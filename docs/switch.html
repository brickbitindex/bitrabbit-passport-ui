Switch - BitRabbit Passport UI

开关

  • 脚本元素
  • 需要JS
  • 锚定选择器: input.bpc-switch[type="checkbox"]

基本使用

<input class="bpc-switch" type="checkbox">

带开关文字：配置项bpc-checked-text, bpc-unchecked-text

<input class="bpc-switch" type="checkbox" bpc-checked-text="开" bpc-unchecked-text="关" checked>

尺寸：配置项bpc-size

<input class="bpc-switch" type="checkbox" bpc-size="large">
<input class="bpc-switch" type="checkbox">
<input class="bpc-switch" type="checkbox" bpc-size="small">

配置

配置项 原生 描述
bpc-size N 尺寸大小，取值空, large, small
bpc-checked-text N 当开关打开时显示的文字
bpc-unchecked-text N 当开关关闭时显示的文字
