Radio - BitRabbit Passport UI

单选以单选框组的形式出现

  • 脚本元素
  • 需要JS
  • 锚定选择器: .bpc-radio-group

基本使用

<span class="bpc-radio-group">
    <input type="radio" name="g1" value="1" bpc-text="A" checked>
    <input type="radio" name="g1" value="2" bpc-text="B">
    <input type="radio" name="g1" value="3" bpc-text="C">
    </span>

禁止状态：配置项disabled

<span class="bpc-radio-group">
    <input type="radio" disabled name="g1" value="1" bpc-text="A" checked>
    <input type="radio" name="g1" value="2" bpc-text="B">
    <input type="radio" name="g1" value="3" bpc-text="C">
    </span>

配置

配置项 原生 描述
bpc-text N 选项显示的标签文字
disabled N 禁用单选框
